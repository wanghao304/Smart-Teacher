## 🎯 核心功能流程

### 首次使用流程
```
登录 → 引导页 → 档案采集(7步20+题) → 导师创建(3种模式) → 主界面
```

### 日常使用流程
```
主界面 → B区查看数据 → A区查看当前项目 → D区与导师对话
       → C区快捷操作（新建导师/项目，更新资料）
```

### 更新资料流程（已修复）
```
主界面 → 更新资料 → profile-view看板 → 点击编辑单项 → 实时保存
```

### 创建导师流程（已优化）
```
选择模式 → 名人模式：选10位之一 → AI分析方法论 → 创建
        → 自定义模式：填写信息 → AI提炼 → 创建
        → AI推荐模式：查看推荐 → 选择 → 创建
```

### 创建项目流程（全新）
```
选择模式 → 自然语言：描述项目 → AI解析 → 确认创建
        → 结构化：填写表单 → 直接创建
```
