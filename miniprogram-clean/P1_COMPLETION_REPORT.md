## 🔄 完整功能流程

### 创建项目到完成的完整链路
```
1. 创建项目
   → 自然语言描述 OR 结构化填写
   → AI解析/直接创建

2. 添加导师
   → 打开项目详情
   → 点击"添加导师"
   → 选择已有导师

3. 推进项目
   → 导师关系图点击💬
   → 与导师对话（关联项目ID）
   → AI提供项目建议

4. 更新进度
   → 项目详情页
   → 点击"编辑进度"
   → 滑块调整0-100%

5. 项目完成
   → 切换状态为"已完成"
   → 查看项目对话历史
   → 导出聊天记录（未来）
```

### 导师从创建到互动的完整链路
```
1. 创建导师
   → 选择模式（名人/自定义/AI）
   → 填写/选择信息
   → AI分析方法论

2. 查看详情
   → 导师列表点击
   → 查看数据统计
   → 查看关键话题

3. 编辑导师
   → 点击"编辑资料"
   → 修改姓名/领域/背景/方法论
   → 点击头像切换Emoji

4. 使用导师
   → 主界面D区选择导师
   → 发送消息开始对话
   → 或在项目中添加导师
```
