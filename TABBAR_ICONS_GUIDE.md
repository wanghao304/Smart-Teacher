## 📂 图标安装步骤

### 步骤1: 创建目录
```bash
# 在小程序目录下创建
mkdir miniprogram/images
mkdir miniprogram/images/tab
```

### 步骤2: 保存图标文件

我已经生成了4组图标，您可以在artifacts中看到它们。请按照以下步骤操作：

#### 方法A: 从artifacts手动保存（推荐）

1. **查看生成的图标**
   - 在右侧artifacts面板查看4组图标图片

2. **处理每组图标**
   - 每组包含2个图标（灰色默认 + 彩色选中）
   - 使用图片编辑工具（如Photoshop/Figma/在线工具）分离两个图标

3. **保存到指定位置**
   ```
   E:\Smart Teacher\miniprogram\images\tab\
   ```

#### 方法B: 使用现成的图标库（临时方案）

如果需要快速测试，可以临时使用iconFont等图标库的SVG转PNG。

### app.json 配置（已更新）
```json
{
  "tabBar": {
    "color": "#999999",
    "selectedColor": "#6366f1",
    "list": [
      {
        "pagePath": "pages/index/index",
        "text": "首页",
        "iconPath": "images/tab/home.png",
        "selectedIconPath": "images/tab/home-active.png"
      }
      // ... 其他3个
    ]
  }
}
```

## 📞 需要帮助？

如果图标显示有问题：

1. **检查路径**: 确保路径与app.json中一致
2. **检查尺寸**: 必须是81x81或类似尺寸
3. **检查格式**: 必须是PNG格式
4. **清除缓存**: 微信开发者工具 → 清除缓存 → 重新编译
